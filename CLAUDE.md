# CLAUDE.md - Cajun Oppskriftskort Nettside

## Prosjektoversikt

Build a static website featuring Cajun recipes presented as illustrated cards. Users should be able to browse through the recipes using swipe gestures, arrow keys, or clicks. Recipes that reference base recipes (roux, Cajun seasoning, holy trinity) should include clickable links.


All the text displayed on the website should be in Norwegian.
---

## Technical Requirements

### Stack
- **Framework:** Next.js 14+ with App Router (Static Export)
- **Language:** TypeScript (strict mode)
- **Styling:** Tailwind CSS
- **Linting:** ESLint + Prettier
- **Deployment:** Vercel (static export)
